# 🏦 Projeto KBank

Este é meu primeiro projeto desenvolvido em Python, com o objetivo de praticar conceitos fundamentais de programação e lógica.

## 📌 Sobre o projeto

O KBank é um sistema bancário simples que simula operações básicas de uma conta, desenvolvido como parte do meu aprendizado em programação.

## 🚀 Funcionalidades

- Criação e manipulação de conta
- Depósito de valores
- Saque de valores
- Consulta de saldo

## 🛠️ Tecnologias utilizadas

- Python

## 🎯 Objetivo

Este projeto foi criado para reforçar meus conhecimentos em:
- Programação orientada a objetos (POO)
- Estruturas de controle
- Organização de código

## 📚 Status

📌 Em desenvolvimento — melhorias e novas funcionalidades serão adicionadas com o tempo.
